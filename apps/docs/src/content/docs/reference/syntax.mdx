## Pro Tips ðŸ’¡

**1. Quotes vs No Quotes**
- Use quotes when values have spaces or special characters
- Skip quotes for simple identifiers and numbers
- When in doubt, quotes always work!

**2. Template Arrays**
- Define your template once, then focus on data
- Perfect for migrating from CSV or spreadsheet data
- Use `map` to add computed fields or defaults

**3. Organizing Large Configs**
- Split by concern: `database.jsson`, `api.jsson`, `auth.jsson`
- Use a main config file that includes others
- Keep related settings together

**4. Expressions**
- Great for derived values (totals, concatenations)
- Reduces manual calculation errors
- Makes configs self-documenting

**5. Ranges**
- Use for any sequential numeric data
- Combine with `step` for custom increments
- Perfect for port ranges, years, IDs

## Keywords Reference

| Keyword | Description |
|---------|-------------|
| `template` | Defines the structure for array items |
| `map` | Transforms template data with custom logic |
| `include` | Imports another JSSON file |
| `step` | Defines the increment in ranges |
| `true` / `false` | Boolean values |

| `/` | Division |
| `%` | Modulo (remainder) |
| `==` `!=` | Equality / Inequality |
| `>` `<` | Greater / Less than |
