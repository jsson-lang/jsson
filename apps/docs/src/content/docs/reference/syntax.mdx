## Conditional Logic ðŸ”€

JSSON supports powerful conditional logic with comparison operators and ternary expressions.

### Comparison Operators

Compare values to get boolean results:

<Code
  lang="jsson"
  code={`age = 20
isAdult = age >= 18      // true
canRentCar = age >= 25   // false
isTeenager = age >= 13
hasDiscount = age < 12`}
/>

Supported operators: `==`, `!=`, `>`, `<`, `>=`, `<=`

### Ternary Operator

Choose between two values based on a condition:

<Code
  lang="jsson"
  code={`status = age >= 18 ? "adult" : "minor"
access = role == "admin" ? "full" : "limited"`}
/>

You can even nest them for complex logic:

<Code
  lang="jsson"
  code={`salary = level == "senior" ? 120000 :
         level == "mid"    ? 80000  :
                             50000`}
/>

  </TabItem>

**Organization:** Split by concern (`database.jsson`, `api.jsson`). Use main file to include others.

**Expressions:** Great for derived values. Reduces errors, makes configs self-documenting.

**Ranges:** Perfect for sequential data. Use `step` for custom increments.

## Keywords Reference

| Keyword          | Description                                |
| ---------------- | ------------------------------------------ |
| `template`       | Defines the structure for array items      |
| `map`            | Transforms template data with custom logic |
| `include`        | Imports another JSSON file                 |
| `step`           | Defines the increment in ranges            |
| `true` / `false` | Boolean values                             |

| `==` `!=` | Equality / Inequality      |
| `>` `<`   | Greater / Less than        |
| `>=` `<=` | Greater/Less than or equal |
| `? :`     | Ternary conditional        |
