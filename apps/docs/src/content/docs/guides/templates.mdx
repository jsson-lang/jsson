## Basic Syntax

The template syntax has two parts:

1. **Template definition:** `template { field1, field2, field3 }`
2. **Data rows:** One row per object, values separated by commas

<Code lang="jsson" code={`arrayName [
  template { column1, column2, column3 }
  value1, value2, value3
  value4, value5, value6
]`} />

## When to Use Templates

**Use templates when:**
- âœ… You have multiple objects with the same structure
- âœ… You're creating seed data or fixtures
- âœ… You're defining lists of similar items (routes, products, users)
- âœ… You want to avoid repetitive JSON

**Don't use templates when:**
- âŒ You only have 1-2 objects (just use regular objects)
- âŒ Each object has a completely different structure
- âŒ The data is highly nested (use regular objects instead)

## Troubleshooting

### Wrong Number of Values

```
Error: template expects 3 fields, got 2
```

**Solution:** Make sure each row has the same number of values as the template:

<Code lang="jsson" code={`users [
  template { name, age, email }
  JoÃ£o, 25, "joao@test.com"  // âœ… 3 values
  Maria, 30                   // âŒ Only 2 values
]`} />

### Missing Quotes

```
Error: unexpected token
```

**Solution:** Use quotes for values with spaces or special characters:

<Code lang="jsson" code={`users [
  template { name, title }
  JoÃ£o, Software Engineer      // âŒ Space in "Software Engineer"
  JoÃ£o, "Software Engineer"    // âœ… Quoted
]`} />
